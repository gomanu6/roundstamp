# roundstamp
Round Stamp It!


### Stamp and Overwrite
- start with a zip file
    - does not support formats other than .zip

- unzip in a folder
- modify and overwrite
- change destination-filename
- zip into new archive
- delete old archive and folder

### Commants and Utilities used
- stat -c %U,
- stat -c %G
- mkdir
- cp
- zip
